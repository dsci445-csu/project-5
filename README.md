# project-5
Class project for group 5 in DSCI445 @ CSU

- The main file of the project is `Final_Project_Git.Rmd`. This file take the results from others R codes since some algorithms we created to tune the Lasso and SVM models took more than 10 min.  
- In the "Lasso" folder, there are the algorithms created to tune the tenderness models of 3, 14 and 28 days, using both method PCA and the data without any transformation. In those files we tune the model for the best lambda value.  
- In the "SVM" folder we included the codes for tuning the linear, polynomial and radial SVM models. In those files we tune the models to select the best cost and number of principal components of all the type of SVM models, and the degree of the polynomial model.  
- The presentation can be found in file `Presentation.Rmd`
- A summary of the final contribution of each collaborator for the coding and presentation can be found in the "Issues" section.
